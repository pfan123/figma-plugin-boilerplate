# figma-plugin-boilerplate

`figma-plugin-boilerplate` —— 快速开发 Figma plugin 模版样例。

## 安装

```
git clone git@github.com:pfan123/figma-plugin-boilerplate.git
```

### 使用

安装依赖：

```bash
yarn install
```

开发调试：

```bash
yarn run start
```

打包构建：

```bash
yarn run build
```
